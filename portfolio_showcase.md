## Project Description: Next-Gen Engineering Portfolio

**Project Title:** Professional Multidisciplinary Portfolio & CMS
**Role:** Full-Stack Architect & UI/UX Designer
**Tech Stack:** React 19, Vite, Firebase (Firestore, Hosting), Tailwind CSS 4, Three.js, Lucide React.

### Overview
A high-performance, multidisciplinary portfolio platform designed to showcase the intersection of physical engineering and software development. This project goes beyond a static site, featuring a custom-built Content Management System (CMS) that allows for real-time updates without code changes.

### Key Features
*   **Bespoke Admin Dashboard:** Developed a secure administrative area to manage projects, skills, and profile data dynamically via Firebase Firestore.
*   **Immersive UI/UX:** Built an interactive 3D background using **Three.js** and a custom particle engine for a premium, high-tech aesthetic.
*   **Dynamic Project Filtering:** Implemented a robust filtering system for projects across categories like Frontend, Backend, UX/UI, and Engineering.
*   **Rich Media Integration:** Integrated multi-image sliders and YouTube video embeds within detailed project overlays for deep-dive showcases.
*   **Performance First:** Optimized for speed using Vite and React 19, ensuring sub-second load times and smooth transitions across all devices.

### Challenges Overvome
*   **State Management:** Unified local state with real-time Firebase listeners to ensure the UI stays in sync with administrative changes.
*   **Responsive Complexity:** Balanced heavy 3D visualizations with mobile-first performance, utilizing lazy loading and intersection observers.
